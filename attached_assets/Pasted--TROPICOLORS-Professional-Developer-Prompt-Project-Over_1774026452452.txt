# 🔵 TROPICOLORS - Professional Developer Prompt

## Project Overview

**Project Name:** TropicColors SPA  
**Type:** E-commerce Single Page Application  
**Business:** Mexican company specializing in artificial food colorants  
**Technology Stack:** React + TypeScript + Express.js + PostgreSQL + Stripe + Firebase

---

## 📋 Project Summary

TropicColors is an e-commerce platform for selling artificial food colorants in Mexico. The application includes:

- **Public Storefront:** Product showcase with cart and checkout
- **Payment Processing:** Stripe integration for card payments
- **Order Management:** Admin dashboard with sales analytics
- **Invoice System:** PDF invoice generation and email delivery
- **Notifications:** Email notifications for orders (both customer and admin)
- **WhatsApp Integration:** Quote requests via WhatsApp

## 💳 Payment Configuration

### Stripe Integration
- **Secret Key:** STRIPE_SECRET_KEY (sk_test_... / sk_live_...)
- **Webhook Secret:** STRIPE_WEBHOOK_SECRET (whsec_...)
- **Currency:** MXN (Mexican Pesos)
- **Webhook Events:** checkout.session.completed, payment_intent.succeeded

### Environment Variables Required
```env
# Database
DATABASE_URL=postgresql://...

# Stripe
STRIPE_SECRET_KEY=sk_test_...
STRIPE_WEBHOOK_SECRET=whsec_...

# Email (Brevo/SMTP)
EMAIL_SMTP_HOST=smtp-relay.brevo.com
EMAIL_SMTP_PORT=587
EMAIL_SMTP_USER=your-brevo-key
EMAIL_SMTP_PASS=your-brevo-password
EMAIL_FROM=noreply@tropicolors.mx

# Twilio (Optional - for SMS)
TWILIO_ACCOUNT_SID=...
TWILIO_AUTH_TOKEN=...
TWILIO_PHONE_NUMBER=...

# Security
SESSION_SECRET=random-32-character-string
ALLOWED_ORIGINS=https://yourdomain.com

# App
NODE_ENV=production
PORT=3000
BASE_URL=https://yourdomain.com
```

## 🔧 Configuración Requerida (Debes hacer esto)

### Paso 1: Obtener API Key correcta de Brevo

1. Ve a https://app.brevo.com
2. Inicia sesión
3. Ve a **Configuración → Claves API**
4. Crea una nueva clave (comienza con `xkeysib-`)
5. Agrega al archivo `.env`:

```env
EMAIL_SMTP_HOST=smtp-relay.brevo.com
EMAIL_SMTP_PORT=587
EMAIL_SMTP_USER=xkeysib-tu-clave-aqui
EMAIL_SMTP_PASS=tu-password-smtp
```

### Paso 2: Desactivar recibos automáticos de Stripe

1. Ve a https://dashboard.stripe.com/settings/emails
2. Desactiva "Enviar correos electrónicos de recibo"
3. Esto permitirá que tu correo HTML personalizado llegue al cliente

### Paso 3: Configurar tu correo para recibir notificaciones

En tu archivo `.env`:

```env
# Tu correo personal para recibir notificaciones de pedidos
EMAIL_TO=tu-correo-personal@gmail.com

# Correo del negocio
EMAIL_FROM=noreply@tropicolors.mx
```

### Paso 4: Configurar Base de Datos (Opcional - ya funciona con Firebase)

Si quieres usar PostgreSQL en lugar de Firebase:

1. Crea una cuenta en https://neon.tech
2. Crea un nuevo proyecto
3. Copia la URL de conexión
4. Agrega al `.env`:

```env
DATABASE_URL=postgresql://usuario:password@host.neon.tech/db?sslmode=require
```

### Paso 5: Variables de entorno completas

```env
# ================================================
# STRIPE (Pagos)
# ================================================
STRIPE_SECRET_KEY=sk_test_...   # Obtén en https://dashboard.stripe.com/apikeys
STRIPE_WEBHOOK_SECRET=whsec_...  # Obtén en https://dashboard.stripe.com/webhooks

# ================================================
# EMAIL (Brevo SMTP)
# ================================================
EMAIL_SMTP_HOST=smtp-relay.brevo.com
EMAIL_SMTP_PORT=587
EMAIL_SMTP_USER=xkeysib-tu-api-key
EMAIL_SMTP_PASS=tu-password
EMAIL_FROM=noreply@tropicolors.mx
EMAIL_TO=tu-correo-personal@gmail.com  # <-- IMPORTANTE: Tu correo para pedidos

# ================================================
# SEGURIDAD
# ================================================
SESSION_SECRET=cadena-random-de-32-caracteres
ALLOWED_ORIGINS=https://tu-dominio.com

# ================================================
# APP
# ================================================
NODE_ENV=production
PORT=3000
BASE_URL=https://tu-dominio.com
```
